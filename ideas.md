# KORE7 Studio - Conceitos de Design

## Contexto
O KORE7 é um studio de treinamento personalizado de alta qualidade, localizado em São Luís, MA. Busca transmitir profissionalismo, exclusividade, segurança e acompanhamento próximo. O público-alvo são pessoas que buscam fugir de academias lotadas e querem metodologia integrada (condicionamento, mobilidade, prevenção de lesões).

---

## Conceito 1: Minimalismo Premium com Acentos Dinâmicos
**Probabilidade: 0.08**

### Design Movement
Minimalismo contemporâneo com influências do design escandinavo, enfatizando clareza, espaço negativo e tipografia assertiva.

### Core Principles
1. **Clareza Radical**: Cada elemento tem propósito; nada é decorativo
2. **Hierarquia Tipográfica Forte**: Tipografia como protagonista visual
3. **Espaço Negativo Generoso**: Respiro visual para transmitir exclusividade
4. **Acentos Estratégicos**: Cores vibrantes em pontos-chave para quebrar monotonia

### Color Philosophy
- **Base**: Branco puro (fundo) + Preto profundo (texto)
- **Acentos**: Laranja-queimado vibrante (#D84315) para CTAs e destaques
- **Secundário**: Cinza quente (suave) para elementos terciários
- **Reasoning**: Laranja transmite energia, movimento e transformação; contrasta com frieza do minimalismo

### Layout Paradigm
- Composição assimétrica com grid invisível
- Seções com largura variável (não centradas uniformemente)
- Imagens grandes e ousadas alternando entre esquerda/direita
- Tipografia em escala dramática (títulos muito grandes, corpo muito pequeno)

### Signature Elements
1. **Linhas Geométricas Sutis**: Divisores angulares entre seções
2. **Tipografia Oversized**: Títulos ocupando 60-80% da altura da viewport
3. **Cards Flutuantes**: Elementos com sombra mínima, flutuando sobre fundo

### Interaction Philosophy
- Transições suaves e longas (300-400ms) para sensação de luxo
- Hover effects sutis: mudança de cor + elevação leve
- Scroll-triggered animations que revelam conteúdo gradualmente

### Animation
- Fade-in em cascata para listas de serviços
- Parallax leve em imagens de fundo (velocidade reduzida)
- Hover: aumento de escala 1.02x + mudança de cor do acento

### Typography System
- **Display**: Playfair Display 700 (títulos principais)
- **Heading**: Poppins 600 (subtítulos)
- **Body**: Inter 400/500 (corpo de texto)
- **Hierarchy**: Títulos 3.5rem → 2rem → 1.25rem → 1rem

## Conceito 2: Brutalism Digital com Energia Orgânica
**Probabilidade: 0.07**

### Design Movement
Brutalism digital combinado com formas orgânicas e tipografia experimental, criando tensão visual que transmite força e autenticidade.

### Core Principles
1. **Formas Ousadas e Irregulares**: Nada perfeitamente alinhado
2. **Tipografia Experimental**: Fontes com personalidade forte
3. **Contraste Extremo**: Luz vs. escuro, grande vs. pequeno
4. **Materialidade Digital**: Bordas ásperas, texturas visíveis

### Color Philosophy
- **Base**: Cinza escuro quase preto (#1a1a1a) com branco puro
- **Acentos Primários**: Amarelo elétrico (#FFD700) + Verde-limão (#BFFF00)
- **Reasoning**: Cores vibrantes contra fundo escuro criam energia bruta; transmitem dinamismo e força

### Layout Paradigm
- Composição caótica controlada com elementos sobrepostos
- Tipografia em ângulos (rotações de 2-5 graus)
- Imagens com bordas cortadas ou mascaradas irregularmente
- Blocos de cor sólida interrompendo o fluxo visual

### Signature Elements
1. **Formas Orgânicas Mascaradas**: Imagens em shapes irregulares
2. **Tipografia Rotacionada**: Títulos com rotação sutil
3. **Bordas Ásperas**: Elementos com clip-path irregulares

### Interaction Philosophy
- Animações abruptas e diretas (150-200ms)
- Hover effects que mudam cor radicalmente
- Cliques que provocam "saltos" visuais

### Animation
- Entrada rápida com scale + rotate simultâneos
- Hover: mudança de cor + rotação adicional (5-10 graus)
- Scroll: elementos aparecem com bounce suave

### Typography System
- **Display**: Space Mono 700 (títulos, monoespacial)
- **Heading**: Bebas Neue 400 (subtítulos, todas maiúsculas)
- **Body**: IBM Plex Sans 400/500 (corpo, legível)
- **Hierarchy**: Títulos 4rem → 2.5rem → 1.5rem → 1rem

## Conceito 3: Elegância Corporativa com Toque Humano
**Probabilidade: 0.06**

### Design Movement
Design corporativo moderno com influências de luxury branding, mantendo acessibilidade e calor humano através de fotografia autêntica e espaçamento respirável.

### Core Principles
1. **Profissionalismo Acessível**: Corporativo mas não frio
2. **Fotografia Autêntica**: Pessoas reais em momentos genuínos
3. **Paleta Sofisticada**: Cores que transmitem confiança e exclusividade
4. **Tipografia Clássica**: Fontes atemporais e legíveis

### Color Philosophy
- **Base**: Branco + Azul-marinho profundo (#1a3a52)
- **Acentos**: Ouro suave (#C9A961) + Teal (#2B7A8F)
- **Secundário**: Cinza quente para transições
- **Reasoning**: Azul-marinho transmite confiança; ouro adiciona luxo; teal humaniza com modernidade

### Layout Paradigm
- Grid simétrico mas com breathing room generoso
- Seções alternando imagem-texto com proporções 60/40
- Cards com bordas suaves e sombras profundas
- Tipografia centralizada em seções-chave

### Signature Elements
1. **Cartões Elevados**: Cards com sombra profunda e hover effect
2. **Bordas Suaves**: Radius consistente em todos os elementos
3. **Ícones Customizados**: Ícones com estilo próprio (não genéricos)

### Interaction Philosophy
- Transições elegantes (250-350ms) com easing suave
- Hover effects que elevam elementos (box-shadow aumenta)
- Scroll-triggered reveals com fade + slide

### Animation
- Fade-in suave com slide vertical ao entrar em viewport
- Hover: elevação (shadow aumenta) + cor muda sutilmente
- Scroll: parallax suave em imagens de background

### Typography System
- **Display**: Lora 700 (títulos, serif elegante)
- **Heading**: Montserrat 600 (subtítulos)
- **Body**: Source Sans Pro 400/500 (corpo, legível)
- **Hierarchy**: Títulos 3.2rem → 1.8rem → 1.2rem → 1rem

---

## Decisão Final
**Conceito Selecionado: Elegância Corporativa com Toque Humano**

Este conceito foi escolhido porque:
1. Transmite profissionalismo e confiança (essencial para um studio de treinamento)
2. Mantém acessibilidade e calor humano (alinhado com "atendimento próximo")
3. A paleta sofisticada (azul-marinho + ouro + teal) diferencia o KORE7 de academias genéricas
4. Fotografia autêntica reforça a proposta de exclusividade e qualidade
5. É versátil para mobile e desktop sem perder elegância
